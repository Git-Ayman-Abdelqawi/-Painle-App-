🖌️ مشروع الرسم (Painle App)
📌 الفكرة
المشروع عبارة عن تطبيق رسم بسيط باستخدام C# Windows Forms. الهدف هو محاكاة أدوات الرسم الأساسية مثل القلم، الممحاة، الأشكال الهندسية، أداة الملء، مع إمكانية التراجع (Undo) وحفظ العمل كصورة. الفكرة جاءت لتطبيق عملي لمفاهيم OOP و Graphics في C#.

📝 التخطيط
تحديد الأدوات الأساسية:

قلم للرسم الحر.

ممحاة لمسح أجزاء.

أشكال هندسية (دائرة، مستطيل، خط، مثلث).

أداة ملء (Fill).

اختيار الألوان وحجم القلم.

زر Undo للتراجع.

حفظ الصورة كملف (PNG, JPG, BMP).

فتح لوحة جديدة (New Canvas).

تخطيط الواجهة (UI):

استخدام PictureBox كلوحة الرسم.

أزرار للأدوات.

ComboBox لاختيار حجم القلم.

أزرار للألوان (Palette).

MenuStrip يحتوي على خيارات (Save, New).

تخطيط الكود:

تعريف متغيرات أساسية: Bitmap, Graphics, Pen, Stack<Bitmap>.

التعامل مع أحداث الماوس: MouseDown, MouseMove, MouseUp, MouseClick.

دوال مساعدة: Undo(), SaveState(), chickPicx() للملء.

⚙️ التنفيذ خطوة بخطوة
1
إنشاء اللوحة
تهيئة الـ Bitmap و Graphics كلوحة رسم.

إنشاء Bitmap بحجم الـ PictureBox

ربطه بـ Graphics

مسح الخلفية بلون افتراضي (Wheat)

2
إضافة أدوات الرسم
تحديد القلم والممحاة والأشكال.

تعريف Pen للرسم

تعريف Pen للممحاة بلون الخلفية

استخدام متغير select لتحديد الأداة

3
التعامل مع أحداث الماوس
تفعيل الرسم عند الضغط والسحب.

MouseDown: بدء الرسم وتخزين نقطة البداية

MouseMove: رسم حسب الأداة المختارة

MouseUp: إنهاء العملية وحفظ نسخة في المكدس

4
تنفيذ أداة الملء
استخدام خوارزمية Flood Fill.

دالة chickPicx لتغيير اللون

استخدام Stack<Point> لتتبع النقاط

تحديث الـ Bitmap وإعادة العرض

5
إضافة ميزة Undo
التراجع عن آخر عملية.

تخزين نسخة من الـ Bitmap في undoStack

عند الضغط على Undo: استرجاع آخر نسخة

تحديث الـ PictureBox

6
حفظ الصورة
استخدام SaveFileDialog لحفظ العمل.

تحديد الصيغ (PNG, JPG, BMP)

استدعاء bm.Clone لحفظ الجزء المعروض

استخدام btm.Save(sfd.FileName)

7
فتح لوحة جديدة
إعادة تهيئة اللوحة للرسم من جديد.

مسح الخلفية بلون Wheat

إعادة تعيين الأداة المختارة

تحديث الـ PictureBox
